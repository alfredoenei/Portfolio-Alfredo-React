## 🇪🇸 Versión en Español

### 🚀 Sobre el Proyecto
Este no es solo un sitio web personal; es una **Single Page Application (SPA)** de alto rendimiento diseñada para demostrar mis habilidades como **Desarrollador Full Stack** y **Arquitecto de Software**. Con una estética inspirada en terminales modernos y el estilo *Glassmorphism*, este portfolio prioriza la fluidez, la limpieza de código y la interactividad.

### ✨ Características de Ingeniería
- **Arquitectura de Componentes**: Diseño modular estricto con separación de capas entre UI, lógica de negocio y constantes.
- **Optimización de Rendimiento**: Implementación de `React.lazy` y `Suspense` para carga diferida de secciones, garantizando un *First Contentful Paint* ultra-rápido.
- **Sistema de Internacionalización (i18n)**: Motor de traducción propio basado en **React Context API** con persistencia en `localStorage`.
- **Diseño Glassmorphism Premium**: Interfaz moderna con fondos translúcidos, desenfoques profundos y micro-animaciones reactivas mediante **Framer Motion**.
- **Gestión de Proyectos Dinámica**: Sistema centralizado de datos que facilita la expansión y actualización del contenido.

### 🛠️ Stack Tecnológico
- **Core**: React 19, Vite 7.
- **Estilos**: Custom CSS con utilidades de Bootstrap 5.
- **Animaciones**: Framer Motion (para transiciones suaves y efectos de entrada).
- **Icons**: Bootstrap Icons.

### 📦 Configuración Local
1. Clona el repositorio.
2. Ejecuta `npm install`.
3. Lanza el entorno de desarrollo con `npm run dev`.
4. Construye para producción con `npm run build`.

## 🇺🇸 English Version

### 🚀 About the Project
This is more than just a personal website; it's a high-performance **Single Page Application (SPA)** designed to showcase my skills as a **Full Stack Developer** and **Software Architect**. Featuring a modern terminal-inspired aesthetic and *Glassmorphism* style, this portfolio prioritizes fluidity, clean code, and interactivity.

### ✨ Engineering Key Features
- **Component-Based Architecture**: Strict modular design with clear layer separation between UI, business logic, and constants.
- **Performance Optimized**: Implementation of `React.lazy` and `Suspense` for lazy loading, ensuring an ultra-fast *First Contentful Paint*.
- **Internationalization System (i18n)**: Custom-built translation engine based on **React Context API** with `localStorage` persistence.
- **Premium Glassmorphism Design**: Modern UI with translucent backgrounds, deep blurs, and reactive micro-animations powered by **Framer Motion**.
- **Dynamic Project Management**: Centralized data system for easy content expansion and updates.

### 🛠️ Tech Stack
- **Core**: React 19, Vite 7.
- **Styling**: Custom CSS with Bootstrap 5 utility classes.
- **Animations**: Framer Motion (for smooth transitions and entry effects).
- **Icons**: Bootstrap Icons.

### 📦 Local Setup
1. Clone the repository.
2. Run `npm install`.
3. Start the development server with `npm run dev`.
4. Build for production with `npm run build`.
